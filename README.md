# linkmus-monitor
Система мониторинга гетерогенной сети на Go. Кроссплатформенные агенты (Linux/Windows) со сбором метрик (CPU, RAM, Disk, Net) и специфичных служб Windows (FSRM, RDP). Архитектура Push-модели, мастер-сервер с REST API и дашбордом на React. Курсовой проект по дисциплине «Телекоммуникационные технологии в отраслях транспортно-дорожного комплекса»
